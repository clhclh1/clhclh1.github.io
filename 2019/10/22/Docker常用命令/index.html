Docker常用命令

2019年10月22日 278 字 大概 1 分钟 Docker

Docker部署靶场常用命令

Docker基础命令

  1. Docker启动：systemctl start docker
  2. 守护进程重启:sudo systemctl daemon-reload
  3. 重启docker服务:systemctl restart docker
  4. 重启docker服务:sudo service docker restart
  5. 关闭docker:service docker stop
  6. 关闭docker:systemctl stop docker

    容器相关命令

  7. 搜索镜像：docker search sqli-labs
  8. 下载所需镜像：docker pull acgpiano/sqli-labs
  9. 运行相关镜像(例)：docker run -dt --name sqli-labs -p 8080:80 --rm acgpiano/sqli-labs
  10. 查看有哪些镜像:docker images
  11. 查看正在运行的容器：docker ps
  12. 进入相关容器：docker exec -it ID /bin/bash
  13. 启动一个或多少已经被停止的容器:docker start ID
  14. 停止一个运行中的容器:docker stop ID
  15. 重启容器：docker restart ID
  16. 删除容器：docker rm ID
  17. 删除镜像：docker rmi IMAGE
  18. 拷贝文件：docker cp 目录名 容器名：目录名
  19. 容器保存为镜像：docker commit 容器名 镜像名
  20. 镜像备份：docker save -o 文件名 镜像名
  21. 恢复镜像：docker load -i 文件名
  22. 启用docker daemon-reload：systemctl daemon-reload+重启dockersystemctl restart docker.service
